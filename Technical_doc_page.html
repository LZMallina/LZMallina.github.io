Where To

In order for html to recognize a JavaScript, the code must be surrounded by the script element. The script can be located in the head or body of the HTML page. It can also stored externally on a separate file or website. Advantage of storing the function externally is easier to read codes and faster loading of the HTML page.

Output

You can display JavaScript using properties: innerHTML, document.write(), window.alert(), or console.log().

innerHTML property must link to an id within the HTML page. An example:

< body>
< h>Example of innerHTML < /h>
< p id="demo">< /p>
< script> document.getElementById("demo").innerHTML="Example of innerHTML property."; < /scrip>
< /body>

Example of document.write(). This property should only be use for testing. Using this property after HTML load will erase other html elements.

< script> document.write( 5+6); < /script>

Window.alert allows a new window popout with your desired message. An example:

< script>window.alert(5+6); < /script>

console.log() is only useful for debugging. An example:

< script> console.log(5+6); < /script>

Statements

A computer program is a list of instructions followed by the computer. In JavaScript programming, this list of instructions are called "statemnets". Statements are comprise of value, operators, expressions, keywords, and comments. An example of a statement is document.getElementById("demo").innerHTML="Example of innerHTML property.";

Satements are exercuted on the order it is written and always end with a ;. Statements can also be group together by declaring a function using {}; For example:

function myFunction() {
document.getElementById("demo1").innerHTML = "Hello Dolly!";
document.getElementById("demo2").innerHTML = "How are you?";
}

Syntax

JavaScript syntax is a set of rules. Syntax is defined by fixed value or variable values. Example of fixed value is a set of numbers (with or without decimals) or strings (text messages), which can be written with single or double quotes.

syntax variables varies based on declarations, operators, identifiers, keywords, and commands. To declare a variable, you would use the property var. Variables are used together with operators (+,-,=,*,/) to compute a value. For example:

< script>
var x, y;
x = 5;
y = 6;
documnent.getElementById('demo'),innerHTML = (x+y); < /script>
Comments

Sometimes you need to document something to organize your thoughts. The way to do that is by using the // for single line comments or /* and */ for multi-line comments. For example:

< body>
< h1 id="myH">< /h1>
< p id="myP">< /p>
< script>
/* The code below will change the heading with id = "myH" and the paragraph with id = "myP" */
document.getElementById("myH").innerHTML = "JavaScript Comments";
document.getElementById("myP").innerHTML = "My first paragraph.";
< /script>
< /body>
